# סימולציות והדגמות - מערכת ניהול מבנים ונכסים

## קבצים זמינים

### 1. MANUAL_SIMULATIONS.html
קובץ HTML אינטראקטיבי עם סימולציות של:
- יצירת מבנה חדש
- יצירת נכס חדש
- תהליך ייבוא שלד
- תפריט ניווט
- לשוניות לפי אזורי מס
- אימות בזמן אמת
- טבלת נכסים אינטראקטיבית

**איך להשתמש:**
1. פתח את הקובץ `MANUAL_SIMULATIONS.html` בדפדפן
2. התנסה בכל הסימולציות
3. לחץ על הכפתורים כדי לראות את התהליכים בפעולה

### 2. קובצי דוגמה בתיקיית `examples/`

קובץ דוגמה לייבוא מבנים:

#### sample_skeleton_import.xlsx / CSV

## איך להשתמש בקובצי הדוגמה

1. **לבדיקת ייבוא:**
   - הורד את קובץ הדוגמה הרלוונטי
   - פתח את המערכת
   - לך לתפריט "ייבוא שלד" או "ייבוא מלא"
   - טען את קובץ הדוגמה
   - בדוק שהנתונים נטענים נכון

2. **כדי להבין את הפורמט:**
   - פתח את קובצי ה-CSV ב-Excel
   - בדוק את שמות העמודות
   - ודא שהנתונים בפורמט הנכון

3. **כדי ליצור קבצים משלך:**
   - העתק את קובץ הדוגמה
   - החלף את הנתונים בנתונים שלך
   - שמור את הקובץ
   - טען למערכת

## הערות חשובות

- כל קובצי הדוגמה כוללים רק נתונים לדוגמה - לא לטעון אותם למערכת הפעילה
- שמות העמודות חייבים להתאים בדיוק (case-sensitive)
- ודא שהפורמט של הנתונים נכון (מספרים, תאריכים, וכו')
