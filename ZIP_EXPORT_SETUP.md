# הוראות התקנה - יצוא ל-ZIP

## התקנת חבילות נדרשות

לפני השימוש ביצוא ל-ZIP, צריך להתקין את החבילה `jszip`:

```bash
npm install
```

זה יתקין את החבילות הבאות:
- `jszip` - ליצירת קבצי ZIP
- `@types/jszip` - TypeScript types עבור JSZip

## מה השתנה:

1. **נוצר קובץ ZIP אחד** במקום מספר קבצים נפרדים
2. **הקובץ ZIP מכיל:**
   - `שליחת_נתונים_YYYYMMDD.xlsx` - קובץ Excel עם נתוני הנכסים
   - `רשימת_קבצים_YYYYMMDD.xlsx` - קובץ Excel עם רשימת הקבצים (asset_id, payer_id, file_name)
   - תיקיית `files/` עם כל קבצי התמונות/PDF מאורגנים לפי asset_id:
     - `files/{assetId}/{filename}`

## מבנה קובץ ה-ZIP:

```
שליחת_נתונים_YYYYMMDD.zip
├── שליחת_נתונים_YYYYMMDD.xlsx
├── רשימת_קבצים_YYYYMMDD.xlsx
└── files/
    ├── 908742001/
    │   ├── file1.pdf
    │   └── file2.jpg
    ├── 908742002/
    │   └── file3.pdf
    └── ...
```

## שימוש:

1. לחץ על "שליחת נתונים לעירייה" ב-BuildingsList
2. המערכת תיצור קובץ ZIP אחד עם כל הקבצים
3. הקובץ יורד אוטומטית

## הערות:

- אם אין קבצים לנכסים, הקובץ ZIP יכלול רק את שני קבצי ה-Excel
- הקבצים מאורגנים בתיקיות לפי asset_id כדי להקל על הניווט
- כל הקבצים נדחסים ב-ZIP כדי לחסוך מקום
