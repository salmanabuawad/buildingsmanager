# מדריך משתמש – מערכת ניהול נכסים

מדריך עזר למשתמשי המערכת. בעקביות לחץ **F1** בכל מסך להצגת עזרה מותאמת להקשר.

---

## תפקידים

| תפקיד   | גישה |
|--------|------|
| **מנהל** | גישה מלאה – הגדרות, משתמשים, איפוס שליחה, משימות ביקורת |
| **משתמש** | עריכה וניהול נכסים ומבנים |
| **פקח** | משימות ביקורת שהוקצו אליו בלבד |

---

## מבנים

- **מבנה חדש:** תפריט מבנים → מבנה חדש → מספר בניין, אזור מס, כתובת (חובה) → שמור.
- **עריכה:** לחיצה על תא → הזן ערך → Enter או לחיצה מחוץ לתא.
- **ייבוא מבנים:** נכסים → ייבוא File → העלה CSV/Excel → מיפוי עמודות (מספר בניין, אזור מס, כתובת) → ייבא.

---

## נכסים

- **עריכת נכס:** לחץ על מספר בניין → ברשימת הנכסים לחץ על תא → הזן ערך → Enter.
- **הוספת נכס:** "הוסף נכס" → מזהה משלם, תאריך מדידה, סוג נכס (חובה).
- **שרטוט/קבצים:** נכס → "הוסף תמונה/וידאו" → בחר נכס → העלה קובץ (תמונות מתכווצות ל־30KB).
- **ייבוא נכסים:** נכסים → ייבוא מלא / ייבוא שלד → העלה Excel → מיפוי → ייבא.

---

## שליחת נתונים לעירייה

1. **ממבנים:** בחר מבנים/נכסים → "שליחת נתונים לעירייה".
2. **מנכסים שנמדדו ולא נשלחו:** נכסים שנמדדו ולא נשלחו → סינון → "שליחת נתונים לעירייה".

**מה נשלח:**
- קובץ **ZIP** הכולל: קובץ Excel עם רשימת הנכסים + קבצי הנכסים (תמונות/מסמכים).
- **מייל** למפעילים ולמנהלים לפי אזורי מס; לכל נמען קובץ מצורף עם שם תקין.
- לאחר שליחה מוצלחת הנכסים מסומנים כ"נשלחו" ולא יישלחו שוב עד איפוס.

**דרישות:** מפעילים/מנהלים מוגדרים בהגדרות מערכת; SMTP מוגדר ופעיל.

---

## איפוס שליחת נתונים

**מנהלים בלבד:** ניהול → פעולות מנהל → איפוס שליחת נתונים מתאריך X → אשר.  
נכסים שנשלחו בתאריך זה יסומנו שוב כ"לא נשלחו" ויוכלו להישלח מחדש.

---

## העברת שטחים

1. רשימת נכסים → סמן נכסים (או הזן מספרי נכס).
2. "העברת שטחים" → בחר מבנה יעד.
3. הזן שטח (חלקי) או העבר מלא → אשר.
4. **נכס 999 (שטח משותף):** העברת שטחים → "הוסף נכס 999" → מזהה, שטח, הערה → שמור.

---

## פיזור שטח משותף

- **שטח משותף מגורים/עסקים:** רשימת נכסים → לשונית לפי אזור מס → הזן שטח משותף במבנה → "פזר" → "שמור הכל".
- **היסטוריית פיזור:** "היסטוריית פיזור" – צפייה בפרטי פיזורים; לחיצה על שורה = לפני/אחרי.
- **חניה:** הזן "שטח חניה משותף" ו"מספר יחידות חניה" במבנה; הגדר סוגי נכס עם "שימוש בשטח חניה משותף"; הזן יחידות חניה לכל נכס חניה → "פזר" → "שמור הכל".

---

## משימות ביקורת

- **יצירת משימה:** ניהול → פעולות מנהל → משימות ביקורת → "משימה חדשה" → מבנה (חובה), נכסים (אופציונלי), פקח, עדיפות → שמור. הפקח מקבל אימייל.
- **פקח – מילוי:** משימות ביקורת → "קבל משימה" → מלא דוח → העלה תמונות/וידאו לנכסים → "שלח לאישור".
- **מנהל – אישור/החזרה:** משימה "ממתין לאישור" → "אשר משימה" או "החזר לפקח" (עם הערה). "ביטול משימה" – סטטוס בוטל.
- **קישור גישה ללא התחברות:** משימה → "צור קישור גישה" → העתק/שלח לינק.

---

## העתקה והדבקה

- **בגרידים:** בחר תא → Ctrl+C → תא יעד → Ctrl+V. עובד במבנים, נכסים, סוגי נכס.
- **העתקת סוג נכס:** רשימת נכסים → "סמן כמקור" על נכס מקור → "העתק סוג נכס" על נכס יעד (סוג + תת-סוגים).

---

## חיפוש נכס

נכסים → חיפוש נכס → הזן מזהה נכס או מזהה משלם → Enter → לחץ על תוצאה לפתיחת פרטי הנכס.

---

## הגדרות מערכת (מנהלים)

- **ניהול משתמשים:** הגדרות מערכת → ניהול משתמשים → הוסף/ערוך/שנה סיסמה/מחק.
- **דוא"ל (SMTP):** הגדרות דוא"ל → שרת, פורט, פרטי התחברות, כתובת שולח → "שליחת מייל בדיקה".
- **מפעילים/מנהלים:** מפעילים / מנהלים → שם, אימייל, אזורי מס (פסיקים). יקבלו מייל רק על נכסים באזורים שלהם.
- **הגדרות שדות:** הגדרות שדות → בחר גריד → רוחב, שם, סיכה, נראות, סדר → שמור.
- **סוגי נכס:** הגדרות מערכת → סוגי נכסים – קוד, תיאור, אזור, דגלים, הגבלות שטח.
- **רשימת כתובות:** רשימת כתובות – סמל רחוב, שם רחוב (למיפוי במבנים).

---

## ממשק

- **בהירות/גודל גופן:** אייקון הגדרות בראש → בהירות (בהיר/רגיל/כהה/ניגודיות גבוהה) | גודל גופן (קטן/רגיל/גדול).
- **רענון:** כפתור "רענן" בראש – טוען מחדש מהשרת. מומלץ אחרי ייבוא או שינויים של משתמש אחר.
